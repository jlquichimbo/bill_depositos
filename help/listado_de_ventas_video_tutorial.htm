Boton de busqueda

Debemos hacer click en el boton de busqueda toda vez que se haya hecho algun filtro de busqueda, sino por defecto el sistema busca todas las que existan..Una vez con el listado de facturas y dependiendo del tipo de privilegios que se ten ga se podra realizar varias operaciones a las mismas como :
 
 
  • Firmar las facturas, se la certifica y firma para la facturacion Electronica.
  • De el listado de facturas se puede escoger un cliente para modificar datos (si tiene privilegio para hacer)
  • Modificar Nombre de Vendedor y Tecnico
  • Abrir para realizar Nota de credito
  • Abrir factura para modificar productos y precios si esta en pendiente/reserva.
  • Abrir para anular factura
  • Abrir para tomar serie de productos de la factura.
  • Entre otras cosas mas como exportar el listado de ventas, se realiza la exportacion en un archivo de excel o similar, ademas se puede imprimir el listado seleccionado.
