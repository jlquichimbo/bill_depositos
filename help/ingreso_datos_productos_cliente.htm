Ingreso Datos_Productos_Cliente

Buscar Productos o servicios Por nombre

Permite digitar al menos 4 caracteres que identifiquen a un producto o servicios, mostrara una serie de opciones y permite escojer, sino se sabe o se tiene el codigo del producto se puede hacer click en la opcion 2 para ir a un listado mas completo de productos o servicios

Busqueda completa de Productos o servicios

Al hacer click en el carrito de compras se abra una ventana con un listado completo de productos o servcios para una busqueda completa del producto facturar, una vez que lo identificamos lo podemos seleccionar para se carge sino tambien podemos digital el codigo el campo cod.Producto pra cargar a la facturacion.

Ingreso codigo de Productos o servicios

si se maneja un apolitica de codigos de barras u otro codigo se puede ingresar directamente el codigo del producto para hacer que la facturacion sea mucho mas rapida que buscar manualmente un  producto

Busqueda de cliente por nombre

El sistema permite la busqueda de un cliente por nombre , se debe ingresar al menos 4 caracteres del nombre del cliente para que se proceda a buscar, sino tambien se puede hacer click en la opcion de busqueda para que se desplaye un listado completo de clientes.

Busqueda de clientes

Esta opcion permite buscar clientes de un listado completo de clientes en donde se puede seleccionar directamente a quien se le va a facturar

Busueda de clientes por ID

La busqueda de clientes tambien se puede hacer por su certificado de identidad en este caso puede ser por el numero de pasaporte, cedula ID, automaticamente se cargara el cliente , sino entonces pedira crearlo directamente con 4 campos necesarios, nombres, direccion , telefono y email.
